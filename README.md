# Global-Cybersecurity-Threat-Intelligence-Dashboard
##  Overview

This project is an end-to-end Tableau dashboard developed to analyze global cybersecurity threats and provide actionable threat intelligence insights. The analysis focuses on cyber attack patterns, financial impact, affected industries, attack sources, vulnerabilities, and incident response efficiency across multiple countries from 2015–2024.

The dashboard helps organizations understand emerging cyber risks and supports data-driven cybersecurity decision-making.

## Project Objectives

* Analyze global cyber attack trends by year, country, and industry

* Identify the most common cyber attack types

* Evaluate financial and operational impact of attacks

* Analyze security vulnerabilities contributing to incidents

* Measure incident resolution performance

* Support cybersecurity risk management through interactive visual analytics

## Dataset Information

* 3,000 cybersecurity incident records

* Time Period: 2015–2024

* 10 Countries analyzed

* 7 Industries covered

* 6 Major attack types including:

1. DDoS

2. Phishing

3. Ransomware

4. Malware

5. SQL Injection

6. Man-in-the-Middle Attacks

## Key Insights

* DDoS and Phishing emerged as the most dominant cyber attack types.

* IT and Banking sectors experienced the highest cyber exposure.

* Zero-day vulnerabilities were the most exploited security weakness.

* Nation-state actors represented a major identified attack source.

* Average financial loss exceeded $50M per incident.

* Cyber attacks remained consistently high across the decade.
